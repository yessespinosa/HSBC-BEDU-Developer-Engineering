#### Instrucciones
##### Considerando tu proyecto seleccionado, distingue los elementos de Scrum que revisamos en el Work y completa las tablas y ejercicios que se muestran a continuación.

##### 1. Identifica qué ceremonias puedes implementar en tu equipo de trabajo y enlístalas.
| Ceremonias |	Justificación |
| ----- | ---- | 
| Planeación |	Es en donde se pondrían en claro cuales son los objetivos del proyecto |
| Diaria |	Verificar qué se hizo, cómo y cuál fue el impedimento |
| Seguimiento |	Los avances logrados |
| Retrospectiva	| Cuáles fueron las estrategias que funcionaron y cuales no |
	
##### 2. Identifica qué roles centrales puedes asignar en tu proyecto.
| Roles |	Justificación | 
| ----- | ---- |
| Dueño del proyecto |	Es el jefe jerárquico y el que conoce los requerimientos del producto |
| Scrum master |	Traducir los requerimientos a la implementación al proyecto |
| Equipo |	Aplicación de requerimientos |

	
##### 3. Selecciona qué valores de Scrum son los que identificas en tu equipo de trabajo.
| Valores |	Justificación |
| ----- | ---- |
| Foco	| Se tiene claro qué es lo que se tiene que entregar |
| Compromiso |	Existe la disposición y disponibilidad de entrega |
| Respeto	| Cordialidad dentro del equipo |

	
##### 4. ¿Qué principios destacas más en tu equipo de trabajo? y ¿por qué?
