## Módulo 1: Desarrollo de software agil

# Objetivos

Aprender a diseñar un pipeline de despliegue que cumpla con todas las fases del despliegue: creación, construcción, pruebas y despliegue de manera automática al detectar cambios.
Esto bajo una propuesta de gestión bajo Scrum o prácticas ágiles.
