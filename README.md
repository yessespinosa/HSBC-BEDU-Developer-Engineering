## HSBC-BEDU Developer Engineering

# Talent lab

# Objetivos

- Conocer y diferenciar entre los marcos de trabajo ágiles (Lean, Kanban y Scrum) 
- Comprender el concepto de filosofía DevOps, explicando los términos de integración delivery continua
- Comprender pipelines de CI/CD con Jenkins y hacer configuraciones básicas 
- Usar Git como herramienta de administración de código y colaboración 
- Utilizar la terminal de sistemas operativos Linux
- Utilizar spring boot para el desarrollo de servicios RESTful
- Emplear JPA para la conexión de bases de datos
- Realizar pruebas unitarias con HUnit
- Comprender y utilizar el patrón MVC con Spring
- Comprender el concepto de microservicio y aplicarlo con Java y Spring boot
- Comprender el paradigma de programación reactiva y streams
- Comprender, diferenciar y explicar patrones de diseño 
- Optimizar el diseño de sus componentes de software empleando patrones de diseño
